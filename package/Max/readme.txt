The Max folder contains various folders provided as examples of the 
Interlude Score Viewer use. They require Max version 5.

You should launch the INScoreViewer application before opening the 
enclosed .maxpat files. In case you don't, most of the patches have a reload 
button to reload the score.

Patches at the root of the folder:
	- inscore.maxpat	: a general score viewer controler.
	- filePlayer.maxpat : a subpatch of inscore.maxpat.
	- effect.maxpat 	: an effects controler (blur, shadow, colorize).

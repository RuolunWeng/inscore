---------------------
  inscore2inscore90 
---------------------

inscore2inscore90 is a tool designed to convert inscore files to the new file format.
The new file format is introduced with INScore version 0.90
The changes are the following:
  - semicolon added at the end of each message
  - '//' comment not supported any more
  - %' comment char replaced by '!'
  - variables declaration
  - single quote support for strings

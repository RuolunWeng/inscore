14.2 Feeding and composing FAUST processors

A FAUST processor accepts float values as input, which are taken as interleaved data.

From composition viewpoint, a FAUST processor is a parallel signal which dimension is the number of output channels. Thus, a FAUST processor can be used like any parallel signal.

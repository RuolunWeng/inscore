15.4 Differences to version 0.42

  • identifier specification change (section 1 p.).
  • new application hello and defaultShow messages (section 2 p.).
  • new load and save messages (sections 2 p. and 4 p.).
  • click and select messages (section 4.3 p.):
    • rightbottom and leftbottom modes renamed to bottomright and bottomleft
    • new center mode for the click message
    • query mode sent back with the reply both for click and select messages
  • new file, html and htmlf types for the set message (section 6 p.).
  • get syntax change for the scene (section p.).
  • fileWatcher messages completely redesigned (section 12.1 p.).
  • mappings can be identified by names (section 9 p.).
  • rect, ellipse, curve, line and polygon object support graphic to relative-time mapping
  • new synchronization modes for Guido scores: voice1, voice2, ... , staff1, staff2, ... , system, page (section 11.1 p.).
  • Guido mapping manages repeat bars.
  • Graphic signals messages design (section 13.2 p.).

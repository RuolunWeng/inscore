Chapter 4
Miscellaneous messages

The following messages are supported by all the objects. They are detailled in specialized sections.

miscMsgs
  • eval: evaluates a list of messages in the context of the receiver object. See section 15.2 p.124 for more details.
  • watch: used to manage the object interaction with various events. See section 14 p.111 for more details.
  • push, pop: saves and restores the object interaction state. See section 14.3 p.121 for more details.
  • map: used to describe the relations between graphic and time spaces. See section 11 p.74 for more details.

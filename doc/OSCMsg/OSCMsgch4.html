Chapter 4
Miscellaneous messages

The following messages are supported by all the objects. They are detailled in specialized sections.

miscMsgs
  • eval: evaluates a list of messages in the context of the receiver object. See section 16.2 p.142 for more details.
  • watch: used to manage the object interaction with various events. See section 15 p.128 for more details.
  • push, pop: saves and restores the object interaction state. See section 15.3 p.139 for more details.
  • map: used to describe the relations between graphic and time spaces. See section 12 p.90 for more details.

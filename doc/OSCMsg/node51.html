15.1 Differences to version 0.63

  • drag and drop files support in INScore viewer
  • interaction variables extension: $sx, $sy variables added to support scene coordinate space (see section 14.2 p.).
  • automatic range mapping for $x, $y variables.
  • new $self and $scene variables in the address field (see section 14.4 p.).
  • OSC identifiers characters set extended with '_' and '-' (see section 1 p.).
  • support for multiple scenes: new, del and foreground messages (see section 3 p.).
  • load message supported at scene level (see section 3 p.).
  • get watch implemented.
  • watch message without argument to clear all the watched events (see section 14.2 p.).
  • order of rendering and width, height update corrected (may lead to incorrect rendering)
  • bug with gmn score corrected: missing update for page, columns and rows changes.
  • package delivered with the Guido Engine version 1.41 that corrects minimum staves distance and incorrect mapping when optimum page fill is off.

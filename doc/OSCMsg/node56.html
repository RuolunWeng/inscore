15.1 Differences to version 0.77

  • guido system map extended: supports flat map or subdivided map (see section 11.1 p.).
  • new shear and rotate transformations messages (see section 4.1.4 p.).
  • new rename message to change an object name (and thus its OSC address) (see section 4 p.).
  • relaxed bool parameter policy: objects accept float values for bool parameters
  • automatic numbering of exports when destination file is not completely specified i.e. no name, no extension. (see section 4 p.).
  • quantification introduced to $date variable (see section 14.3 p.).
  • reset message addressed to a scene clears the scene rootPath

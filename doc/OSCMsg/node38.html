11.3 Score queries



  • pageCount: gives the score page count.
  • pageDate: gives the time location of a page, which number is given as argument. The system reply has the form pageDate pageNum date where pageNum is the request page number given as argument and date is the page time location expressed as a ra tional number.

Note that a score object duration is automatically set to the score duration. It can be retrieved using a get duration message.

17.14 Differences to version 0.77

  • guido system map extended: supports flat map or subdivided map (see section p.).
  • new shear and rotate transformations messages (see section 2.2 p.).
  • new rename message to change an object name (and thus its OSC address) (see section 2 p.).
  • relaxed bool parameter policy: objects accept float values for bool parameters
  • automatic numbering of exports when destination file is not completely specified i.e. no name, no extension. (see section 2 p.).
  • quantification introduced to $date variable (see section 13.2 p.).
  • reset message addressed to a scene clears the scene rootPath
